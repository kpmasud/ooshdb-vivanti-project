# 🍕 OOSHDB: PostgreSQL Practice Project for Food Retail System

This project simulates a realistic database for a Lebanese food business named **Ooshman** operating in Australia. It's built to support hands-on learning and practice with:

- Learn about databases
- SQL & PostgreSQL commands  
- Schema design and ERD modelling  
- Query writing and business logic  
- Data manipulation and injection  
- Dashboard and analytics  

> ⚠️ **Note:** All data in this project is synthetic, except for the menu items, shops, delivery methods, and payment methods, which were collected from Ooshman's real website.

These files will be continuously updated as I learn more:
